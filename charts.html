Navigating the charts

This report features interactive charts. Here are some tips and an example to get familiar with the interactivity:

  • Filtering/selecting Data: Some charts allow for adding & removing data by clicking on the legend item. Some charts also have dropdown filters above the chart.
  • Zooming: Some charts allow for zooming in on dates. Click on the chart area and drag to the left or right to select a custom date range.
